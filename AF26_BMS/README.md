## 改动
相对上一个版本，主要进行了如下改动：
1. 调整了整体的文件结构，使所有文件均位于 ../HAL_BMS_Modified/Core 下，所有由用户创建的文件均加了 “-user” 后缀便于辨认，这样做的好处是在包含头文件时可以不用操心头文件路径相关的问题。在调整文件结构的同时，尽量将头文件的包含该到了相应的".c"文件中，同时去除了一些不必要的头文件依赖，使得头文件依赖关系更加清晰；
2. 重写了任务调度算法（主要在“os-user.c”中），将基于数组的任务调度算法改为基于链表。这使得任务调度算法有了在运行时动态添加/删除任务的能力；
3. 在重写任务调度算法的同时，为任务函数增加了可选的传参以及返回值接口，增加了任务函数调用的灵活性，希望能够借此减少全局变量的使用，以减少各文件之间的耦合，使得代码结构更加清晰。

后续将根据需要继续优化主控代码。
